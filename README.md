# 🐾 Petify Frontend

A modern frontend application for the Petify platform, providing an interactive user interface for pet management, e-commerce, and appointment scheduling.

---

## 🚀 Features

* 🔐 Authentication (Login / Register / Forgot Password)
* 🐶 Pet Profiles Management
* 🛒 E-commerce (Browse products, Cart, Checkout)
* 📅 Appointment Booking System
* 👤 Admin Dashboard

---

## 🛠️ Tech Stack

* React.js
* React Router
* React Hook Form
* Axios
* Context API (state management)
* Tailwind CSS
* DaisyUI

---

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/omnjja/Petify-Front-end.git
```

### 2. Install dependencies

```bash
npm install
```


### 3. Run the project

```bash
npm run dev
```

---

## 🔗 API Integration

This frontend connects to the backend API:

* Base URL: `http://localhost:8080`
